# Проектная работа по "Основы работы с данными для ИИ" Южного федерального университета

Данный проект посвящён анализу датасета, содержащего информацию о трендах зимней моды за период 2023–2025 годов. Целью исследования является выявление ключевых закономерностей, популярных категорий одежды, цветовых решений, материалов, а также изменений потребительских предпочтений во времени.

## Цели и задачи
- Исследование исходного датасета
- Предобработка данных (удаление пропущенных значений и дубликатов)
- Анализ полученного датасета
- Визуализация выявленных закономерностей

## Стек технологий
- **Jupyter Notebook** – интерактивная среда для анализа данных
- **Yandex DataLens** – платформа для создания аналитических дашбордов
- **Python** – язык программирования для обработки данных
- **Pandas** – библиотека для манипуляции с табличными данными
- **NumPy** – библиотека для математических вычислений
- **Matplotlib** – инструмент для построения графиков
- **Seaborn** – библиотека для статистической визуализации
